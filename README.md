# Harsh Lagwal | AI Engineer Portfolio

A modern, high-performance personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. This portfolio showcases my work in Generative AI, Machine Learning, and Full Stack Development.

## 🚀 Features

- **Dynamic Hero Section**: Modern UI with smooth animations and responsive design.
- **Professional Timeline**: An interactive experience section showing my career growth.
- **Verified Certifications**: A visual certifications showcase with real-time PDF previews (integrated with Google Drive).
- **Responsive & Premium UI**: Fully optimized for Mobile, Tablet, and Desktop with a custom dark/light theme system.
- **Performance Optimized**: Lazy-loaded sections, non-blocking font loading, and high-performance animations.

## 🛠️ Tech Stack

- **Core**: React 18, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theming**: Context API (Light/Dark mode support)

## 📦 Deployment

This project is ready to be deployed on platforms like Netlify or Vercel.

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
