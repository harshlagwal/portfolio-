---

## 🏗️ System Architecture & Design Philosophy

This portfolio is built on a modular architecture designed for extensibility and rapid iteration.

- **Component-Driven Development**: Leveraging Atomic Design principles to ensure reusability and maintainability.
- **Micro-Animation Engine**: Powered by `framer-motion` for meaningful transitions that enhance UX without compromising performance.
- **Glassmorphic UI**: A custom design system implementing sophisticated backdrop filters, deep shadows, and dynamic gradient borders for a premium aesthetic.
- **Adaptive Resilience**: Advanced responsive logic ensuring pixel-perfect layouts across viewport ranges (320px to 4K).

## ⚡ Core Technical Features

### 🧠 AI Engineering Showcase
- **Technical Timeline**: A data-driven chronological mapping of internships and contributions at organizations like **Upto Skills**, **eDC IIT Delhi**, and **Edunet Foundation**.
- **Integrated Credentials**: Custom modal system for real-time document previewing (experience letters, certifications) via Google Drive API integration.

### 🛠️ Developer Experience (DX)
- **Vite Ecosystem**: Sub-second HMR and optimized build pipelines.
- **Theme Orchestration**: Context-based theme engine with persistent state management and zero-layout-shift (ZLS) transitions.
- **Modern Logic**: Custom hooks for scroll tracking, intersection observers for reveal animations, and optimized asset handling.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/harshlagwal/portfolio-.git

# Navigate to the project directory
cd portfolio-

# Install dependencies
npm install

# Launch development server with HMR
npm run dev
```

### Production Build
```bash
# Generate optimized production bundle
npm run build

# Preview production build locally
npm run preview
```
